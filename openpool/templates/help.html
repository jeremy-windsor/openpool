Help

OpenPool is a local-first pool chemistry logbook and calculator for tracking tests, doses, maintenance, and water balance. It keeps the day-to-day UI simple while exposing the same pool data through a generated API for backups, dashboards, and local automation.

API Reference

The authoritative, always-current API reference is generated from this running OpenPool app at /docs and /openapi.json. Use the generated OpenAPI when you need exact request and response schemas for the version you are running.

Interactive Docs ReDoc OpenAPI JSON

Repository docs such as README.md, docs/formulas.md, and docs/deployment.md live with the project source in GitHub; this app does not render repository markdown.

API Quickstart

Health

curl {{ base_url }}/api/health

List pools

curl {{ base_url }}/api/pools

Create a reading

curl -X POST {{ base_url }}/api/pools/{{ pool_id }}/readings \
  -H 'Content-Type: application/json' \
  -d '{"fc": 4, "cc": 0, "ph": 7.6, "ta": 80, "ch": 350, "cya": 40}'
